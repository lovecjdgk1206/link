CAM-01 · SYSTEM

게임 시스템

01 / 06

비밀매칭게임은 현실 기반 회원제 커뮤니티 위에서 돌아가는 시즌제 게임입니다. 시즌1의 이름은 스토커 게임, 포지션은 스토커와 피해자 둘뿐입니다.

참가 방식

  • 참가는 자발적이며, 참가 이유는 서로 묻지 않습니다.
  • 참가자는 직업·학교·기존 인간관계를 그대로 유지한 채 게임을 병행합니다.
  • LINK(일상형) — 평소 일상 위에 게임이 스며드는 방식
  • 리조트(합숙형) — 시즌 기간 동안 회원 전용 리조트에 상주
  • 리조트형은 개인실 제공, 외출 자유, 숙박·휴식은 리조트 내에서만 진행

  • 선택지: 기존 관계 유지 / 관계 역전 / 새로운 관계 / 게임 관계 종료

CAM-02 · MISSION

미션

02 / 06

미션은 지금 상황과 관계, 진행도, 성향을 반영해 매번 다르게 생성됩니다. 같은 미션은 반복되지 않습니다.

TYPE 01 · 공통

공통미션

주 1회, 기한 7일. 포지션별로 다른 내용이 개별 공지되며 반대 포지션에게는 비공개입니다.

TYPE 02 · 개인

개인미션

운영진이 개별로 전달합니다. 기한은 따로 없고, 완료 전까지는 다음 개인미션이 생성되지 않습니다.

TYPE 03 · 릴레이

릴레이미션

참가자가 다른 참가자를 지정해 24시간 안에 이어갑니다. 완료 후엔 다음 참가자를 다시 지정할 수 있습니다.

이 외에 운영진 판단으로 즉시 발생하는 이벤트미션이 있습니다. 같은 장소에 여러 참가자가 모였을 때 참여는 자유, 완료는 즉시, 보상도 즉시 지급됩니다.
  • 미션을 실패하거나 거부하면 벌칙이 따를 수 있습니다.
  • 벌칙은 포인트를 깎지 않습니다.
  • 원치 않으면 블랙마켓에서 거부권을 구매할 수 있습니다.
  • 미션·벌칙의 성패는 스토커·피해자 진행에 영향을 주지 않습니다.

CAM-06 · RESORT

리조트 구역별 이용안내

06 / 06

리조트 합숙형을 선택하면 시즌 기간 동안 회원 전용 부지에 상주합니다. 낮은 각자의 일상, 저녁부터는 리조트로 돌아오는 흐름입니다.

개인실

  • 전 참가자에게 개인실 1인 1실 배정
  • 외출은 허가 절차 없이 자유
  • 숙식·취침·휴식은 리조트 안에서만 진행
  • 낮에는 기존 출근·통학 일상 유지, 저녁~밤은 리조트 복귀 필요

라운지

  • 회원 전용 메인 교류 공간, 개방 시간 18:00 ~ 04:00
  • 펜트하우스·식사 공간·바 테이블·무대·게임 공간·수영장으로 연결된 구조
  • 대화·식사·추리·정보교환 자유롭게 진행
  • 주 1~2회 공식모임, 그 외 시간은 자유 이용
